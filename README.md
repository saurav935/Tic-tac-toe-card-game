This is a simple tic-tac-toe game built using Python

To run this program offline, by using terminal or command prompt

- Fork this repo

- Clone the repo

- git clone [url]

- replace [ url ] with the link copied from clone option

- Change Directory

- cd tic-tac-toe

To run program

python3 [file_name].py
